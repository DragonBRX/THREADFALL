# THREADFALL

**THREADFALL** é um jogo de ação e aventura com elementos de plataforma, ambientado em um mundo pós-apocalíptico onde a morte renasce para enfrentar deuses caídos.

## 📁 Estrutura do Repositório

- `assets/`: Contém todos os recursos visuais (sprites, cenários, parallax).
- `docs/`: Documentação técnica, GDD (Game Design Document) e planos de desenvolvimento.
- `src/`: Código fonte do jogo.
- `index.html`: Ponto de entrada para a versão web.
- `firebase.json`: Configuração para deploy no Google Antigravity (Firebase Hosting).

## 🚀 Como Executar

O projeto está configurado para ser executado via navegador. Para desenvolvimento local, você pode usar qualquer servidor estático (como o Live Server do VS Code).

## 🌐 Deploy

O jogo está preparado para deploy no **Firebase Hosting**.
