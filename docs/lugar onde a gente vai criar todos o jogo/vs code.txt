
O que é o VS Code
O Visual Studio Code é um editor de código-fonte gratuito, leve e multiplataforma (Windows, Linux e macOS), desenvolvido pela Microsoft.
Ele não é uma IDE completa por padrão, mas pode se transformar em uma através de extensões.
Principais características
Leve e rápido
Altamente personalizável
Código aberto (open-source)
Suporte a várias linguagens
Integração nativa com Git
Grande ecossistema de extensões

Extensões
As extensões são o principal diferencial do VS Code e permitem transformá-lo em uma IDE completa.
Exemplos populares:
Python – Debug, lint e autocomplete
C/C++ – IntelliSense e depuração
ESLint / Prettier – Qualidade de código Web
Live Server – Atualização automática no navegador
Prettier – Formatação de código
ESLint – Análise de erros
GitLens – Recursos avançados de Git

Resumo final
O Visual Studio Code é um editor extremamente versátil, rápido e poderoso, capaz de atender desde iniciantes até profissionais. Com extensões, ele se adapta a praticamente qualquer tipo de projeto, sendo hoje uma das ferramentas mais importantes do desenvolvimento de software.
